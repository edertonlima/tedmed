Display a nice custom fixed menu on your webiste

== Description ==

Display a nice custom fixed menu on your webiste. You can choose :
- menu position (top, bottom, left or right)
- font size
- icons size
- for each menu : background and icon color

A Pro version is available with more features : http://www.info-d-74.com/produit/nice-fixed-menu-pro-plugin-wordpress/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the menu named "Nice Fixed Menu"
